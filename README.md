# nano-blue
**A (bluebird promisified) minimalistic couchdb driver for node.js**

`nano-blue` wraps [nano][] in [bluebird-flavored][] promises.

[nano]: http://github.com/dscape/nano
[bluebird-flavored]: http://github.com/petkaantonov/bluebird
